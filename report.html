Report

Overall

Guidelines in the assignment specification dictates that our e-commerce website should consist out of XML Content page, registration form, slide show and this report. It also needs to have responsive design and be accessible on all devices and browsers. Every page on the website consists out of header, content, footer and navigation bar.

Design wise I will use just the two colours in order to make pages simple and minimalistic. Black and white combination will be seen throughout the whole website, same as consistent styling elements. Headers will have the name of the site which is also a link same as icons for sign-in page and cart page. Contents of the pages depends on the purpose of the given page and the data that is displayed. In footer will be displayed information about developer and link to home page. Every page is accessible through the navigation bar.

“Sign-in” page consists of the form with wide range of input elements such as name, email, password, date of birth. There are verification of required fields and input data type. There is also an “cart” page where is form for payment details.

Slideshow page is made by putting images to container with the help of JS functions. In the top container corner is number of the current slide image and description of that image. Right next to it you can find image thumbnails through which you can change slides. Buttons for next and previous slide are on the right and left edge of image container.

Responsive design is achieved by use of media queries and website works on all devices and display sizes. In the time of writing this report there was no problems with responsiveness.
